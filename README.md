# agro
 🌱 AgroLink is a simple trust-based platform that connects farmers directly with buyers, eliminating middlemen and improving profit margins for farmers.It enables farmers to list their crops and buyers to discover and contact them easily.

Farmers often depend on intermediaries to sell their product, leading to:
- Reduced profits
- Lack of price transparency
- Limited market access

AgroLink provides a direct farmer-to-buyer connection:

- Farmers can list crop details (Name, Crop, Location)
- Buyers can view listings and contact farmers
- Data is stored locally using browser LocalStorage (no backend required)

🧩 Features
- 👨‍🌾 Farmer Data Entry System
- 🛒 Buyer Listing Dashboard
- 💾 LocalStorage Data Persistence
- 📜 Submission History (Admin View)
- 📱 Responsive Design
- ✨ Smooth UI Animations
